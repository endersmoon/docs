# Icons

Our design system uses [Lucide](https://lucide.dev/icons/) as our primary icon library. Lucide provides a beautiful collection of over 1,400+ carefully crafted SVG icons that are consistent, customizable, and open source.

## Overview

Lucide icons are designed with the following principles:
- **Consistent**: All icons follow the same design language and grid system
- **Customizable**: Easily adjust size, stroke width, and color
- **Accessible**: Built with accessibility in mind
- **Lightweight**: Optimized SVG icons with minimal file sizes
- **Open Source**: Free to use under the ISC License





## Customization

All Lucide icons accept the following props:

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `size` | `number \| string` | `24` | Icon size in pixels |
| `color` | `string` | `currentColor` | Icon color |
| `strokeWidth` | `number \| string` | `2` | Stroke width |
| `absoluteStrokeWidth` | `boolean` | `false` | Whether to use absolute stroke width |

## Design Guidelines

### Sizing

Use consistent icon sizes throughout your application:

- **16px**: Small icons for dense UI elements
- **20px**: Medium icons for buttons and form elements  
- **24px**: Default size for most use cases
- **32px**: Large icons for emphasis
- **48px+**: Extra large icons for empty states or illustrations

### Stroke Width

Maintain consistent stroke width across your interface:

- **1px**: Thin style for subtle elements
- **1.5px**: Light style for secondary elements
- **2px**: Default stroke width (recommended)
- **2.5px**: Medium style for emphasis
- **3px**: Bold style for primary actions
