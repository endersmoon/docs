# Checkbox
Let users select one or more options by using the checkboxes component.

<div className='flex  justify-center items-center h-[200px] w-full border gap-4 border-gray-200 rounded-md mt-3'>
   <div className="flex items-center gap-3">
        <Checkbox id="terms" />
        <Label htmlFor="terms">Accept terms and conditions</Label>
      </div>
</div>

### When to use this component
Use the checkboxes component when you need to help users:

- select multiple options from a list
- toggle a single option on or off


### When not to use this component
Do not use the checkboxes component if users can only choose one option from a selection. 
In this case, use the Radios component.
