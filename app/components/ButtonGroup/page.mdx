### Overview

A button group gives people access to frequently performed, related actions. Use button groups when there's a close relationship between multiple buttons. Common placements of grouped buttons can be found below.

### Variants

We have several button group configurations:

- `Single Button`
- `Double Button - Horizontal`
- `Double Button - Vertical`
- `Triple Button`
- `With Options`

### Usage Guidelines

- **Button groups** should be used when multiple related actions need to be presented together.
- **Maintain hierarchy** by using appropriate button variants within a group (primary for main action, outline/ghost for secondary actions).
- **Limit the number** of buttons in a group to 3-4 for optimal usability.
- **Consistent spacing** between buttons is essential for visual clarity.
- **Consider layout** based on available space and importance of actions (horizontal for equal importance, vertical for limited width).

#### Accessibility

- Ensure sufficient color contrast between buttons and their surroundings.
- Maintain adequate spacing between buttons to prevent accidental clicks.
- Consider keyboard navigation for users who cannot use a mouse.
